installation:
To install all the required features for the frontend, 
1. install NodeJS on your machine from this link:
https://nodejs.org/en
2.After installing NodeJS, run the following command in the terminal:
npm install

run the project:

Navigate to the front project folder and run the following command in the terminal:
npm run dev
*** you should see this link apear: http://localhost:5173/ 


**** super important ******
- Do not run the frontend more than once without stopping the previous instance.
- To stop the running services:
  Press Ctrl + C in the terminal.
